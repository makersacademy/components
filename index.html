MA Style Guide

Consistent styles to provide a limited set of templates for building pages. Includes a CSS framework and some HTML styles.

See it in action

Patterns

Atoms (UI visual_guide) -> Groups (Groups of visual_guide) -> templates (template blocks) -> Pages (groups of template blocks). Interfaces should be built using consistent templates.

Atoms

Atoms are individual UI visual_guide. They can be combined together to make Groups.

See Atoms

Groups

Groups are collections of Atoms. They can be dropped into templates to create Templates.

See Groups

Templates

Templates are collections of Groups. They can be used as templates, and define the high-level structure of Pages.

See Templates
